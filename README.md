
# 🎬 Movie Seat Booking App

A simple **Movie Seat Booking Web App** built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Select a movie
* Select available seats
* See selected seats
* Automatic price calculation
* Book or cancel seats

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)

## 📷 Project Preview

This project simulates a movie theatre seat booking system where users can choose seats and calculate total ticket price.
